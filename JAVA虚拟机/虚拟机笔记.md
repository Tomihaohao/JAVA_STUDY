# 虚拟机

## 基本汇编基础
- 数据传送指令
    ```java
        //将自然数1传送到EAX寄存器
        movl 1,%eax
        //将栈顶数据弹出至EAX寄存器
        pop %eax
    ```
- 算术运算指令
    ```java
        //将自然数3与eax寄存器中的数据累加，并将结果存储进eax中
        add 3,%eax
        //对ebx 寄存器中的数增1
        inc %ebx
    ```
- 逻辑运算指令
    ```java
        //将eax中的数左移1个二进位
        shl %eax,1
        //对al寄存器中的数和操作数进行操作
        and al,00111011B
    ```
- 串指令
    连续空间分配，连续空间取值，传送等。
- 程序转移指令 if...else for while 函数调用等
